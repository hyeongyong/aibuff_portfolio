기술적 문제 해결 사례

1학생부 비교과(세특) 표기 정합성

상황
학생부 교과활동 표에서 일부 과목(AP 미시/거시경제, 프로그래밍)의 세특이 누락되고, 일부(심화영어독해)는 영어·기타에 중복 표기. 과학과제연구 누락·과학탐구실험이 '기타'에 표기되는 문제도 포함.
분석
세특 레코드의 대분류가 사회/과학이 아닌 계열명(경영금융·정보통신)으로 내려와 '사회'.includes() 매칭에서 탈락. 부분일치(숫자 포함 과목)는 광범위 매칭으로 교차 중복 발생.
해결
매칭을 완전일치=과목명 신뢰(대분류 무시) / 부분일치=대분류 검사 유지로 분리. 실데이터 검증 후 화면·인쇄 공통 유틸에 반영해 일관성 확보.

2등급제 전환 · 결측 데이터 시각화

상황
2022 개정 5등급제와 기존 9등급 혼재로 성적 게이지/그래프의 등급 매핑·결측(null) 구간 표기 오류 발생.
분석
학년별 등급 스케일 분기 누락, 데이터 결측 시 차트 라벨이 null로 노출되는 지점 식별.
해결
등급 스케일별 매핑 분기 + 결측 구간 예외 처리로 그래프 정상화.

3교육청 전송 시스템 신규 개발

상황
교육청 계정이 학교별 교과성적 제출 현황을 확인하고 일괄로 내려받을 화면이 부재.
분석
설정관리에 교육청 전송 진입점 신설, 학교별 제출/미제출 상태 집계 및 대용량 엑셀 내보내기 요건 정리.
해결
교육청용 교과성적 조회·제출 현황 화면 개발, 교과 원본 데이터 동시 저장으로 무결성 보완, 전체 교과성적 엑셀 일괄 다운로드 구현.
