# 화면 스크린샷 넣는 곳

포트폴리오(`../index.html`)의 **화면 미리보기** 섹션에 표시될 이미지 폴더입니다.
아래 **정확한 파일명**으로 캡처 이미지를 이 폴더에 저장하면 자동으로 표시됩니다.
(이미지가 없으면 자리표시자가 대신 보입니다.)

| 파일명 | 화면 | 비고 |
|--------|------|------|
| `01-main.png` | 메인 랜딩 · 반응형 | 메인 페이지 |
| `02-login.png` | 로그인 · 인증 / SSO | 로그인 화면 |
| `03-record-eval.png` | AI 학생부 평가 리포트 | 평가/리포트 화면 |
| `04-research.png` | 탐구주제 · 탐구보고서 · 모의면접 | 콘텐츠 화면 |
| `05-subscription.png` | B2B 이용권(구독) 관리 · 견적서 | `/admin/.../subscription` |
| `06-admin.png` | 권한별 메뉴 · 관리자 설정 | 관리자 화면 |

## 캡처 팁
- 권장 비율 **16:10** (예: 1600×1000). 다른 비율도 되지만, 카드가 `object-fit: cover`로 상단 기준 크롭됩니다.
- 파일 형식은 `.png` 권장. `.jpg`를 쓰려면 `index.html`의 `src="images/01-main.png"`를 `.jpg`로 바꿔주세요.
- 개인정보(학생 이름·연락처 등)가 보이면 캡처 전에 가리거나 테스트 데이터로 캡처하세요.
- 캡처가 어려우면 윈도우 `Win + Shift + S`(부분 캡처) 후 저장하면 편합니다.

저장 후 브라우저에서 `index.html`을 새로고침하면 반영됩니다.
